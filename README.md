# vocabulary-extension-server
This extension will extract search meaning into word file, like first on the extension and make the search like "happy" in google search bar then it meaning automatically get saved into word file

# STEP TO MAKE USE OF THIS EXTENSION

STEP 1 : Extension ON
STEP 2 : Search word on google search bar, like "happy" then the hindi translate with word get saved into word file 
STEP 3 : OFF the Extension after finishing the work of translation

# Note : This extension extract the first searched word and store it's hindi meaning in the word file so that ON the extension when you are doing the meaning search else extension will extract first word of all search and save there meaning in the word file.
